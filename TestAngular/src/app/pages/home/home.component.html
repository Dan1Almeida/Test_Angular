Introdução

Este projeto tem como objetivo cadastrar pessoas, utilizando atributos específicos para criar o objeto 'Pessoa'. A partir dessa estrutura, será possível gerenciar o cadastro de contatos, atribuindo-os às respectivas pessoas. Além disso, o sistema permitirá a edição dos dados conforme a necessidade e a visualização de uma lista completa de todas as pessoas cadastradas.

- Iniciando

Como iniciar o projeto?

  • Entre na página "Cadastro".

  • Preencha o formulário.

  • Confirme a requisição.

- Tenho que mudar

Errou ou quer atualizar?

  • Entre na página "Edição".

  • Informe o ID Correspondente.

  • Atualize os dados.

  • Confirme a requisição.

- Fazendo Contato...

Já tem pessoa cadastrada?

  • Entre na página "Contatos".

  • Informe o ID Correspondente.

  • Preencha o formulário.

  • Confirme a requisição.

- Todo Mundo Aqui

Quem está cadastrado?

  • Entre na página "Listagem".

  • Encontre todas as pessoas.

  • Exclua pessoas quando necessário.
